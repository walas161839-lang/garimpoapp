⚖️ Legal & Privacidade
⚠️ Este app é informativo. Consulte sempre um advogado especializado em direito minerário.
🔒 Política de Privacidade

GarimpoApp é um aplicativo por assinatura. O acesso completo requer plano ativo. Esta política descreve como tratamos seus dados.

📱 Dados armazenados localmente

Todos os dados inseridos (produção, diário, acessos, cidades, checklist) ficam somente no seu dispositivo via localStorage. Não enviamos nenhum dado para servidores próprios.

🌐 APIs externas consultadas

O app consulta serviços externos apenas para buscar cotações e clima. Nenhum dado pessoal é enviado:

  • • AwesomeAPI — cotação do ouro e câmbio (economia.awesomeapi.com.br)
  • • Open-Meteo — previsão do tempo (open-meteo.com)
  • • GoldPrice.org / MelhorCambio — referências de cotação
  • • AllOrigins / Corsproxy — proxy CORS para acesso às APIs acima

🚫 O que NÃO fazemos

  • • Não coletamos nome, CPF, localização ou dados pessoais
  • • Não exibimos anúncios
  • • Não compartilhamos dados com terceiros
  • • Não requeremos cadastro ou login

🗑️ Como apagar seus dados

Limpe o cache/dados do navegador ou do app para apagar todos os dados locais. Não há dados em nuvem para excluir.
